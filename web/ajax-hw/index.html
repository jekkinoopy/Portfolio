login.js 
$('#form-login').on('submit', function(e) {
  e.preventDefault();
  $.ajax({
    url: 'login.php',
    type: 'POST',
    data: { user, pass },
    dataType: 'json',
    success: function(res) {
      // res.status === 'ok' → 打卡成功
    }
  });
});

野原家戶籍查詢

資料查詢

「美冴：你查什麼？　小新：查看看我有沒有弟弟。」

search.js 
$.ajax({
  url: 'query.php',
  type: 'GET',
  data: { id, type },
  dataType: 'json',
  beforeSend: function() {
    $('#result').html('查詢中…');
  },
  success: function(res) {
    $('#result').html(res.data);
  }
});
